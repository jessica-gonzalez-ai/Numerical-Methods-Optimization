# Numerical Methods & Mathematical Optimization

A collection of high-precision mathematical algorithms implemented in **Python**. This repository focuses on **Numerical Analysis**, providing tools for root-finding, iterative solvers, and systems of non-linear equations.

##  Algorithms Implemented
This library covers the fundamental methods used in scientific computing and AI optimization:

* **Root-Finding Methods:**
    * **Bisection Method:** Implementation with automated sign change justification and error tolerance handling (`ejercicio1.py` through `ejercicio4.py`).
    * **Newton-Raphson:** High-speed convergence using derivatives.
    * **Secant Method:** Root approximation without requiring the analytical derivative.
    * **Fixed Point Iteration:** Convergence analysis for functional forms.
* **Systems of Equations:**
    * **Multivariate Newton-Raphson:** Solving non-linear systems using Jacobian matrices and `numpy.linalg`.

##  Technical Stack
* **Language:** Python 
* **Libraries:** NumPy, Math.

##  Academic Context
Implementation for the **Numerical Methods** coursework at **Instituto Tecnológico de La Paz**, bridging the gap between pure mathematics and computational efficiency.
